This project is a clean and professional admin dashboard interface that includes:

Sidebar navigation

Light/Dark theme toggle

Interactive charts using ApexCharts

Responsive layout for all screen sizes

Modern UI design

The dashboard can be used as a base template for inventory systems, analytics panels, or management portals.

# Technologies Used

HTML5 – Structure of the dashboard

CSS3 – Styling and responsiveness

JavaScript – Dynamic functionality

ApexCharts – Data visualization

Google Fonts & Material Icons – UI enhancement

# Features

Fully responsive layout

Sidebar open/close functionality

Light and Dark mode toggle

Interactive bar and area charts

Clean and modern UI

Mobile-friendly design

# Learning Outcomes

By working on this project, the following learning outcomes were achieved:

Understanding of Responsive Web Design

Learned how to create layouts that adapt to different screen sizes using CSS Grid and Media Queries.

Implementation of JavaScript DOM Manipulation

Gained experience in handling user interactions such as sidebar toggle and theme switching.

Integration of Third-Party Libraries

Learned to integrate and configure ApexCharts for dynamic data visualization.

Working with Dark and Light Themes

Implemented theme switching using JavaScript and CSS classes with local storage support.
